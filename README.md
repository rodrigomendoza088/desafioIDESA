# desafioIDESA
Desafio IDESA #1

El proyecto trata de una API RESTful de una biblioteca, permite la creacion, lectura, actualizacion y eliminacion (CRUD) de libros y autores; Asi como tambien un sencillo modulo de autenticacion.

## Requisitos
PHP >= 8.2.

MySql.

Laravel 11.
